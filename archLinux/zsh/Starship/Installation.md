starship is customizable prompt for any shell

# Dependencies 
- Nerdfont
# Installation 

### Download & Install Starship
```sh
yay -s starship 
```

### Modify `.zshrc`
```sh
# ~/.zshrc 
eval "$(starship init zsh)"
``` 

### Create Configuration File 
```sh
# ~/.config 
touch ~/.config/starship.toml 
```
### All Done Now
